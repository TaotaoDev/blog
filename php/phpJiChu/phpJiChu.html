# 分离HTML

​ 在基础的时候php和html是混合在一起使用的，而php在解析文件的时候，会寻找开始和结束标记，标记告诉我们何时开始解析，什么时候结束，这种方式的解析可以是PHP嵌入到各种不同的文档中，在开始和结束标记之外的PHP解析器都会忽略

​

<?php
if ($expression) {
    ?>
    <strong>This is true.</strong>
    <?php
} else {
    ?>
    <strong>This is false.</strong>
    <?php
    	}
    ?>

# 强制类型转化

  • （int）、（integer）：转换成整形
  • （float）、（double）、（real）：转换成浮点型
  • （string）：转换成字符串
  • （bool）、（boolean）：转换成布尔类型
  • （array）：转换成数组
  • （object）：转换成对象
<?php
$a_bool = true;
$a_str = "foo";
$an_int = 12;

echo gettype($a_bool); //boolean
echo '<hr>';
echo gettype($an_int); //integer
echo '<hr>';

echo gettype((string) ($a_bool));
echo '<hr>';
echo var_dump((string) ($a_bool));

# 5、break

  • 结束循环
  • 在循环语句中使用 while for switch
  • 可以跳出多层循环
示例1：
for( $int=1; $int<10; $int++){

    var_dump($int);
    echo '<hr>';
}
示例2：
$money = 50000;
switch ($money) {
    case $money >= 50000:
        echo '我要买个华为Mate X2 手机';
        break;
    case $money >= 20000:
        echo '我要买个iphone手机';
        break;
    case $money >= 10000:
        echo '我要买个小米手机';
        break;
    case $money >= 5000:
        echo '我只能买个二手手机';
        break;
    default:
        echo '我啥也买不起，洗洗睡吧';
        break;
}

# 2、什么是函数

  • 函数是一段可以重复执行的代码片断
  • 函数是实现代码复用的重要手段
  • 函数是现代编程语言最重要的基本单元
  • 函数永远是编程的核心工作

# 3、函数的分类

  • 根据函数的提供者(编写者),分为二类
    • 系统函数: 编程语言开发者事先写好提供给开发者直接使用的
    • 自定义函数: 用户根据自身需求，对系统功能进行扩展

# 4、作用域

  • php中, 只有函数作用域和全局作用域
  • 所有函数作用域中的变量，外部不可见
  • 全局作用域声明变量，在函数中是可见的
$name = '灭绝师太';
$school = 'PHP中文网';
// 创建函数
function fun_name()
{
    global $name;
    global $school;
    return '我是：' . $name . '，我来至：' . $school;
}

// 调用函数
echo fun_name();
